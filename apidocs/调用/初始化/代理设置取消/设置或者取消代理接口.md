# 设置或者取消代理接口

## 简要描述
- 设置取消代理接口

## 请求URL
- `http://47.94.7.218:8083/wxwork/setProxy`

## 请求方式
- POST
- ContentType: `application/json`

## 参数
| 参数名 | 必选 | 类型 | 说明 |
| --- | --- | --- | --- |
| uuid | 是 | String | 对应账号操作id |

## 请求示例
```json
{
  "uuid": "f866182dfc70b000bcdfcb2013fddcab",
  "ip": "127.0.0.1",
  "port": 1081,
  "proxyType": "http",
  "passward": "",
  "userName": ""
}
```

## 返回示例
```json
{
  "data": null,
  "errcode": 0,
  "errmsg": "ok"
}
```
