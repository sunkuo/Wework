# 推出群聊

## 简要描述
- 推出群聊

## 请求URL
- `http://47.94.7.218:8083/wxwork/OutRoomReq`

## 请求方式
- POST
- ContentType: `application/json`

## 参数
| 参数名 | 必选 | 类型 | 说明 |
| --- | --- | --- | --- |
| uuid | 是 | String | 每个实例的唯一标识，根据uuid操作具体企业微信 |

## 请求示例
```json
{
  "uuid": "4db64991-8ce3-427b-8186-df8841216c4c",
  "roomid": 10696052955016166
}
```

## 返回示例
```json
{
  "data": {
    "room_id": 10696052955016166,
    "msg_id": 1092779
  },
  "errcode": 0,
  "errmsg": "ok"
}
```
