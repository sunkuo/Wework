# 二维码进群

## 简要描述
- 二维码进群

## 请求URL
- `http://47.94.7.218:8083/wxwork/AgreeCardAddRoom`

## 请求方式
- POST
- ContentType: `application/json`

## 参数
| 参数名 | 必选 | 类型 | 说明 |
| --- | --- | --- | --- |
| uuid | 是 | String | 每个实例的唯一标识，根据uuid操作具体企业微信 |

## 请求示例
```json
{
  "uuid": "3240fde0-45e2-48c0-90e8-cb098d0ebe43",
  "code": "7EefxHYdxxxxxx"
}
```

## 返回示例
```json
{
  "data": {
    "data": {
      "vid": "1084618918xxxxxx",
      "status": "Success"
    }
  },
  "errcode": 0,
  "errmsg": "ok"
}
```
