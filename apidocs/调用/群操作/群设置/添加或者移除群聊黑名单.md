# 添加或者移除群聊黑名单

## 简要描述
- 添加或者移除群聊黑名单

## 请求URL
- `http://47.94.7.218:8083/wxwork/addOrSubRoomBlackList`

## 请求方式
- POST
- ContentType: `application/json`

## 参数
| 参数名 | 必选 | 类型 | 说明 |
| --- | --- | --- | --- |
| uuid | 是 | String | 每个实例的唯一标识，根据uuid操作具体企业微信 |

## 请求示例
```json
{
  "uuid": "625e4c0a0b528c3eb9f1c86dd1b48af5",
  "oprType": 1,
  "blacklist_vid": [7881299838913796]
}
```

## 返回示例
```json
{
  "data": null,
  "errcode": 0,
  "errmsg": "ok"
}
```
