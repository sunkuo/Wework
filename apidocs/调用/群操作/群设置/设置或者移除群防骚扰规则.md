# 设置或者移除群防骚扰规则

## 简要描述
- 设置或者移除群防骚扰规则

## 请求URL
- `http://47.94.7.218:8083/wxwork/setRoomAnti`

## 请求方式
- POST
- ContentType: `application/json`

## 参数
| 参数名 | 必选 | 类型 | 说明 |
| --- | --- | --- | --- |
| uuid | 是 | String | 每个实例的唯一标识，根据uuid操作具体企业微信 |

## 请求示例
```json
{
  "uuid": "dba51402f94c4d9aff9e8a4588a4b689",
  "roomid": 10827700805999285,
  "antiIds": [7485268685645994616]
}
```

## 返回示例
```json
{
  "data": null,
  "errcode": 0,
  "errmsg": "ok"
}
```
