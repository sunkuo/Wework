# 设置群备注

## 简要描述
- 设置群备注

## 请求URL
- `http://47.94.7.218:8083/wxwork/RoomnameRemarkReq`

## 请求方式
- POST
- ContentType: `application/json`

## 参数
| 参数名 | 必选 | 类型 | 说明 |
| --- | --- | --- | --- |
| uuid | 是 | String | 每个实例的唯一标识，根据uuid操作具体企业微信 |

## 请求示例
```json
{
  "uuid": "16xxx",
  "roomid": 10696052955,
  "content": "啊啊啊"
}
```

## 返回示例
```json
{
  "data": null,
  "errcode": 0,
  "errmsg": "ok"
}
```
