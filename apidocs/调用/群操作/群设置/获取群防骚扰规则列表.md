# 获取群防骚扰规则列表

## 简要描述
- 获取群防骚扰规则列表

## 请求URL
- `http://47.94.7.218:8083/wxwork/queryCRMAntiSpamRule`

## 请求方式
- POST
- ContentType: `application/json`

## 参数
| 参数名 | 必选 | 类型 | 说明 |
| --- | --- | --- | --- |
| uuid | 是 | String | 每个实例的唯一标识，根据uuid操作具体企业微信 |

## 请求示例
```json
{
  "uuid": "dba51402f94c4d9aff9e8a4588a4b689",
  "lastIndexInfo": ""
}
```
