# 发送文本表情消息

## 简要描述
- 发送文本表情消息

## 请求URL
- `http://47.94.7.218:8083/wxwork/SendTextAndExpMsg`

## 请求方式
- POST
- ContentType: `application/json`

## 参数
| 参数名 | 必选 | 类型 | 说明 |
| --- | --- | --- | --- |
| uuid | 是 | String | 每个实例的唯一标识，根据uuid操作具体企业微信 |
| send_userid | 是 | long | 要发送的人或群id |
| isRoom | 是 | bool | 是否是群消息 |

## 请求示例
```json
{
  "uuid": "1688853790xxxx",
  "send_userid": 788130255xxxx,
  "isRoom": false,
  "content": [
    { "msgtype": 0, "msg": "你好" },
    { "msgtype": 3, "msg": "[微笑]" },
    { "msgtype": 0, "msg": "哈哈" }
  ]
}
```

## 返回示例
```json
{
  "data": {
    "receiver": 7881302555xxx,
    "sender": 1688853790xxx,
    "at_list": [],
    "sender_name": "",
    "is_room": 0,
    "sendtime": 1678197779,
    "msg_id": 1115653,
    "server_id": 12883461,
    "msgtype": 2,
    "content": "你好[微笑]]哈哈"
  },
  "errcode": 0,
  "errmsg": "ok"
}
```
