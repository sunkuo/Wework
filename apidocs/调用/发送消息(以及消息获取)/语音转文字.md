# 语音转文字

## 简要描述
- 语音转文字

## 请求URL
- `http://47.94.7.218:8083/wxwork/SpeechToTextEntity`

## 请求方式
- POST
- ContentType: `application/json`

## 参数
| 参数名 | 必选 | 类型 | 说明 |
| --- | --- | --- | --- |
| uuid | 是 | String | 每个实例的唯一标识，根据uuid操作具体企业微信 |
| msgid | 是 | int | 要转文字的语音消息id |

## 请求示例
```json
{
  "uuid": "a4ea6a39-4b3a-4098-a250-2a07bef57355",
  "msgid": 1063645
}
```

## 返回示例
```json
{
  "data": {
    "text": "牛牛牛牛。"
  },
  "errcode": 0,
  "errmsg": "ok"
}
```
