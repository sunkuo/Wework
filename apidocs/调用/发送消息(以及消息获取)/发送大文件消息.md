# 发送大文件消息

## 简要描述
- 发送大文件消息，需要走大文件上传，不走 CDN 上传。

## 请求URL
- `http://47.94.7.218:8083/wxwork/SendCDNBigFileMsg`

## 请求方式
- POST
- ContentType: `application/json`

## 参数
| 参数名 | 必选 | 类型 | 说明 |
| --- | --- | --- | --- |
| uuid | 是 | String | 每个实例的唯一标识，根据uuid操作具体企业微信 |
| send_userid | 是 | long | 要发送的人或群id |
| isRoom | 是 | bool | 是否是群消息 |

```json
{
  "uuid": "1688d599424",

  "fileSize": 314065,
  "isRoom": false
}
```
