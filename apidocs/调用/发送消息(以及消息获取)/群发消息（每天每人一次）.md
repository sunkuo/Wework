# 群发消息（每天每人一次）

## 简要描述
- 控制台群发消息

## 请求URL
- `http://47.94.7.218:8083/wxwork/SendGroupsMsg`

## 请求方式
- POST
- ContentType: `application/json`

## 参数
| 参数名 | 必选 | 类型 | 说明 |
| --- | --- | --- | --- |
| uuid | 是 | String | 每个实例的唯一标识，根据uuid操作具体企业微信 |
| vids | 是 | long | 要发送的人或者群idid |
| isroom | 是 | bool | 是否是群消息 |
