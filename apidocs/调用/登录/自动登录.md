# 自动登录

## 简要描述
- 自动登录条件：初始化企业微信接口传递了 `vid`，并通过该 `vid` 获取了对应 `uuid`。
- 如果初始化没有传递 `vid`，则自动登录失败。
- 账号至少成功登录一次后，后期可以持续自动登录。

## 请求URL
- `http://47.94.7.218:8083/wxwork/automaticLogin`

## 请求方式
- POST
- ContentType: `application/json`

## 参数
| 参数名 | 必选 | 类型 | 说明 |
| --- | --- | --- | --- |
| uuid | 是 | String | 每个实例的唯一标识，根据uuid操作具体企业微信 |

## 请求示例
```json
{
  "uuid": "8793dc28-a863-463b-a5d6-ccd369adb859"
}
```

## 返回示例
```json
{
  "data": null,
  "errcode": 0,
  "errmsg": "登陆成功"
}
```
