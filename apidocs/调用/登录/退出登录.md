# 退出登录

## 简要描述
- 退出登录

## 请求URL
- `http://47.94.7.218:8083/wxwork/LoginOut`

## 请求方式
- POST
- ContentType: `application/json`

## 参数
| 参数名 | 必选 | 类型 | 说明 |
| --- | --- | --- | --- |
| uuid | 是 | String | 每个实例的唯一标识，根据uuid操作具体企业微信 |

## 请求示例
```json
{
  "uuid": "f5a22e9b-9664-4250-b40a-08741dba549c"
}
```

## 返回示例
```json
{
  "data": {
    "code": 0,
    "errMsg": "ok",
    "obj": {},
    "pbObj": null
  },
  "errcode": 0,
  "errmsg": "退出成功"
}
```
