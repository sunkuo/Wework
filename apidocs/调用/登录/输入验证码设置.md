# 输入验证码设置

## 简要描述
- 第一次登录的时候需要输入验证码。
- 如果登录过（初始化接口 `vid` 已传值），后续获取二维码通常不需要再输入验证码。

## 请求URL
- `http://47.94.7.218:8083/wxwork/CheckCode`

## 请求方式
- POST
- ContentType: `application/json`

## 参数
| 参数名 | 必选 | 类型 | 说明 |
| --- | --- | --- | --- |
| uuid | 是 | String | 每个实例的唯一标识，根据uuid操作具体企业微信 |

## 请求示例
```json
{
  "uuid": "2b0863724106a1160212bd1ccf025295",
  "qrcodeKey": "B547FC8D049DFA333D89075C04BA9178",
  "code": "369130"
}
```

## 返回示例
```json
{
  "data": null,
  "errcode": 0,
  "errorcode": 0,
  "errmsg": "ok"
}
```

## 注意事项
- 如果调用这个接口出现错误：`qrcode_not need verify`，通常是输入验证码前把手机上的验证码界面提前关闭了。
- 不要提前关闭验证码界面，输入完成后会自动关闭。
